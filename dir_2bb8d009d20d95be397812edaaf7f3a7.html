ResilientDB Orbit

ResilientDB Orbit is a lightweight desktop launcher to spin up ResilientDB and its core services in one click, perfect for handling development, rapid testing, and local experimentation.

Setup

bun i

Useful commands

Start dev server

bun run tauri dev

Build executable

bun run tauri build
